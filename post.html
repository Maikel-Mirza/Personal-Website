Web Hosting Site

This was a project I started when I was learning frontend JavaScript development. I was the sole creator/developer for this project.

The website allows for users to be able to host their own site and was a great learning experience for me.

Allows for users to create an account and login as a user so that they can continue with their web hosting needs

Users have the ability to navigate through a responsive search bar, and the login has authentication requirements to maintain security

I worked on this project for learning purposes but could develop into something with commercial value

Technologies:
  • - JavaScript
  • - React
  • - Express
  • - Mongoose
  • - MongoDB
  • - HTML/CSS
