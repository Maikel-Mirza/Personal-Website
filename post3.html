Student Couch Surfer

This was a project I started when I was learning fullstack development with JavaScript react-native and firebase. I was the sole creator/developer for this project.

The website allows for users to be able to host their own site and was a great learning experience for me.

Allows for users find events and couch surf with other students in the area they are traveling to.

Users have the ability to filter by location, date, and event type.

I worked on this project for learning purposes but could develop into something with commercial value

Technologies:
  • - JavaScript
  • - react-native
  • - React
  • - Firebase
  • - Node
  • - HTML/CSS
